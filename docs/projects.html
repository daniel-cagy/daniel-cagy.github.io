LMCL

A simple litghtweight markup language that transpiles to HTML, designed to look more familiar to programmers. It was built entirely in Rust, and for now can only be used via command line interface (CLI). I'm planning to add a library version in the future so it can be integrated into other projects.

Currently, LMCL supports only strings, static conditionals and basic HTML tags, classes and ids, but I'm working on adding more features like links, assets, lists, and more. It's actually more of an experimental project, of how a markup language could be designed to be more like a programming language.

You can check out the project on GitHub.
